# learn-vue

# Vue CLI

All vue 2 projects need vue cli v4 and all vue 3 projects need vue cli v5.
Install v4: `npm update -g @vue/cli` install v5: `npm install -g @vue/cli@next`, or any version with: `npm install -g @vue/cli@4.0.1 `
